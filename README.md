# Arcadia Admin Panel

**[EN]** Advanced server-side administration tool with GUI, offline player management, and warning system.
**[FR]** Outil d'administration serveur avancé avec GUI, gestion des joueurs hors ligne et système d'avertissement.

---

## 🇺🇸 English (US)

### Features
- **Player GUI**: Manage online and offline players in a paginated interface (InvSee, Clear Inventory, Reset Progress).
- **Offline Data**: View homes and last seen locations (FTB Essentials integration).
- **Warn System**: Warn players with reasons and track their warning history via unified JSON storage (`warns.json`).
- **Moderation**: Teleport to players/homes, Kick, Ban, and Unban directly from the GUI.
- **Permission-Aware**: Buttons (Kick, Ban, TP, etc.) are hidden if you lack the required permissions.
- **Bilingual**: Automatic language detection (English/French) based on client settings.

### Commands
- `/arcadiaadmin panel` *(Op Level 2)*: Opens the main administration interface.
- `/arcadiaadmin reload` *(Op Level 2)*: Reloads the offline player cache and FTB data.
- `/arcadiaadmin warn <player> <reason>` *(Op Level 2)*: Warns a player.
- `/arcadiaadmin warnlist <player>` *(Op Level 2)*: Views a player's warning history.
- `/arcadiaadmin delwarn <player> <index>` *(Op Level 2)*: Deletes a specific warning.
- `/arcadiaadmin checkwarn` *(All)*: Players can check their own warnings.

## 🇫🇷 Français (FR)

### Fonctionnalités
- **GUI Joueurs** : Gérez les joueurs en ligne et hors ligne (InvSee, Vider Inventaire, Reset Progression).
- **Données Hors Ligne** : Consultez les résidences (homes) et la dernière position (intégration FTB Essentials).
- **Système d'Avertissement** : Avertissez les joueurs et suivez leur historique via stockage JSON unifié (`warns.json`).
- **Modération** : Téléportation, Expulsion, Bannissement et Débannissement depuis l'interface.
- **Permissions Dynamiques** : Les boutons (Kick, Ban, TP, etc.) sont cachés si vous n'avez pas la permission.
- **Bilingue** : Détection automatique de la langue (Anglais/Français) selon le client.

### Commandes
- `/arcadiaadmin panel` *(Op Niveau 2)* : Ouvre l'interface d'administration principale.
- `/arcadiaadmin reload` *(Op Niveau 2)* : Recharge le cache des joueurs hors ligne et les données FTB.
- `/arcadiaadmin warn <joueur> <raison>` *(Op Niveau 2)* : Avertit un joueur.
- `/arcadiaadmin warnlist <joueur>` *(Op Niveau 2)* : Affiche l'historique des avertissements d'un joueur.
- `/arcadiaadmin delwarn <joueur> <index>` *(Op Niveau 2)* : Supprime un avertissement spécifique.
- `/arcadiaadmin checkwarn` *(Tous)* : Les joueurs peuvent voir leurs propres avertissements.
